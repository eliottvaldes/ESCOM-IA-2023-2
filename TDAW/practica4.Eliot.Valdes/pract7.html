CPU (Unidad Central de Procesamiento)
- La CPU es el componente principal de una computadora que realiza las operaciones de procesamiento y cálculo. Es el "cerebro" de la computadora.
RAM (Memoria de Acceso Aleatorio)
- La RAM es una memoria temporal de alta velocidad que almacena datos y programas que la CPU está utilizando actualmente. Es volátil y se borra al apagar la computadora.
Disco Duro
- El disco duro es un dispositivo de almacenamiento de datos no volátil que se utiliza para guardar archivos y programas a largo plazo. Almacena datos incluso cuando la computadora está apagada.
Tarjeta Gráfica
- La tarjeta gráfica es un componente que se encarga de procesar y mostrar gráficos en el monitor. Es esencial para la visualización de imágenes y video en la computadora.
Placa Base (Placa Madre)
- La placa base es el circuito principal de la computadora que conecta todos los componentes de hardware, como la CPU, la RAM y las tarjetas de expansión. Actúa como el "esqueleto" de la computadora.
